# DevCom
一个可以共同协作的平台。
在一个平台里，创作、分享、聊天。
## 本地部署：

推荐全局安装 docsify-cli 工具。

```bash

npm i docsify-cli -g

```

克隆仓库：

```bash

git clone https://github.com/mdevcom/mdevcom.github.io.git

```

运行：

```bash

docsify serve mdevcom.github.io

```

默认用的还是系统服务器，可以尝试使用[Firebase](https://console.firebase.google.com)新建项目。
